# 缺口地图

**标签**：**【L2】** 本轮收口 · **【L3】** 联合 L3 · **【L4/seam】** 记录层机制 · **【L2·外阻】** 圈外依赖

## **Entity Resolution**

- **stable 阈值** → **【L2】已收口**（见 `L2_proposals/Entity Resolution__L2提案.md`）
- **stable entity 确认后的 Entity Log 写入动作边界** → **【L2】** ★（写入执行者 / 是否调用专门写入节点 → **【L4/seam】**）
- **entity_resolution_output 字段 schema** → **【L3】**
- **new_stable_entity provenance 字段 schema** → **【L3】**
- **knowledge_summary 冲突修复流程** → **【L4/seam】**

## **Entity Log**

- **entity_record / provenance 字段 schema** → **【L3】**
- **alias_record / Alias 库技术形态** → **【L4】**
- **automation_policy 与 Governance Log projection 边界** → **【L2·外阻】**
- **merge / split 后 alias / case / rule 迁移或阻断语义** → **【L2】** ★（机制 **【L4】**）
- **person entity 与 Profile 边界** → **【L2·外阻】**
- **automation_policy 自动降级 mutation contract** → **【L4/seam】**

## **Alias Log**

- **stable entity 确认后 surface text 是否触发 alias 写入** → 决策 **【L2】** ★；执行 **【L4/seam】**
- **normalization / equivalence 匹配策略** → **【L2】** ★（具体规则 **【L3】**）
- **Alias 冲突 / 多 entity 竞争 / merge-split 迁移语义** → **【L2】** ★（机制 **【L4】**）
- **Alias 对下游的支持范围** → **【L2】**
- **alias_record 字段 schema** → **【L3】**
- **Alias 库技术形态、与 Entity Log 存储边界** → **【L4】**
- **写入责任 / 审批 / 统一 write 机制** → **【L4/seam】**（确认 authority 属于 **【L2】**）
- **Alias mutation 与 Governance audit 边界** → **【L2·外阻】**

## **Case Log**

- **write eligibility** → **【L2】** ★
- **completed / corrected / reversed / split 的 supersession 语义** → **【L2】** ★（机制 **【L4】**）
- **record schema / field / enum / validation** → **【L3】**
- **exact writer / Transaction Log trigger order / 多 log 统一写入** → **【L4/seam】**
- **case-derived rule / automation / entity risk candidate governance contract** → **【L2·外阻】**
- **与 Knowledge Summary 冲突修复** → **【L4/seam】**
