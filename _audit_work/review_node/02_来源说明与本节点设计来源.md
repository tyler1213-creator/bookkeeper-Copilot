## 它在权威分层中的位置（重要）

- 对 **Review 自身功能 / 身份 / 三层结构 / 三铁律 / 决策权限** → 它充当 **owner 已拍板的看法（≈ 模板第 1c 步「用户观点」）**，是 Proposal 可指回的依据之一。
