## 一、行话 / 领域速记
- **地板（floor）**：全系统唯一落盘出口；「任何 durable 写入的最后一脚都踩它」。
- **扩张型变更（expansion-type）**：扩大长期确定性权威的写入 —— rule 升/改/删/降、automation 放宽、merge/split、永久纠错。对立面 = 日常/运行期/候选写入。
- **凭证（credential）**:一条「已记录的会计师签字」的存在性标志，挂在 payload 上；闸门按它开合。
- **有类型的动作**:create / update / append / supersede，校验挂在动作上不挂在地板上。
- **第三类对象**:跨层共享机制，非 node、非 memory layer。
